# D3 visualization with React

Simple demonstration of D3 visualiation using ReactJS.

Clone repo:

``` bash
$ git clone https://github.com/Wyvarn/d3scatterplot.git
```

install dependencies
```
$ cd d3scatterplot
$ npm install

# if using yar instead
$ yarn install
```

Run application

```
$ yarn start

# or with npm
$ npm run start
```

That is it.
